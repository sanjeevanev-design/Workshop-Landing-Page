# Build Better Interfaces — Workshop Landing Page

A premium, high-converting workshop landing page built using **HTML, CSS, and JavaScript**.

Designed for:
- Live workshops
- Online webinars
- Coaching sessions
- Product launches
- Event registrations

---

# Overview

This landing page helps event organizers promote and collect registrations for a live online workshop.

The design focuses on:
- Clear messaging
- Strong CTA placement
- Conversion-focused layout
- Smooth user experience
- Mobile responsiveness
